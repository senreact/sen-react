# Sen React — Discovery

Goal: flesh out the **entire** platform before deciding what ships first. Nothing is parked.

## How to use these documents

1. Open `sen-react-discovery-template.md` — this is the master template. Read it top-to-bottom once before answering, so you can see how the pieces connect.
2. Answer in your own words. If you don't know, write **"not sure yet"** — we'll decide together. There are no wrong answers.
3. Add new questions of your own wherever something is missing. This is your platform.
4. When you're done, Tom will review and we'll translate a cleaned version into French if useful.

## Files

- `sen-react-discovery-template.md` — the master questionnaire in **English**. One document, organised by theme.
- `sen-react-discovery-template-fr.md` — identical template in **French**. Same sections, same numbering — pick whichever language you prefer (or mix).
- `site-scrape-findings.md` — structured extraction of senreact.com (6 projects, 13 trainings, Amadou's bio, partners, brand colours). Source material; **pending Amadou's confirmation that this content is current and reusable** (see `docs/amadou/en/03-final-validation.md` Question 1).
- `decisions-log.md` — running log of decisions locked in (D001–D009), with source and architectural implications.
